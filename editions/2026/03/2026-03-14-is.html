📦Microsoft Fabric February 2026 uppfærsla kemur með víðtækar endurbætur

Mánaðarleg Fabric uppfærsla færir marktækar endurbætur á alla pallinn: Workspace Apps birtast nú í OneLake Catalog (loksins, samræmd uppgötvun), endurbætt útgáfusaga fyrir notebook með mörgum heimildum, VS Code viðbót fyrir vafra á item skilgreiningum, API fyrir GraphQL private link stuðning, og customer managed key dulkóðun fyrir notebook kóða. Auk þess straumlínulagaðar item upplýsingar sem ná út fyrir catalog. Stjórnunar og þróunarupplifun endurbæturnar eru raunverulega gagnlegar—ekki bara demo rugl.

⚡Microsoft BitNet verður vinsælt með 1-bit LLM inference rammakerfi

Opinbert inference rammakerfi Microsoft fyrir 1-bit LLM sprakk á GitHub í dag (2.227 stjörnur á 24 klukkustundum). Þetta er ekki bara rannsóknagrein lengur—þetta er framleiðslutilbúin inference innviði sem gæti lækkað compute kostnað verulega fyrir data teymi sem keyra LLM vinnuálag. Þess virði að fylgjast með ef þú ert að gera AI inference í stórum stíl.

🤖Claude fær 1M context glugga almennt tiltækan

Anthropic gerði 1M context almennt tiltækt fyrir Opus 4.6 og Sonnet 4.6. Þetta er gríðarlegt fyrir data greiningu workflows—þú getur nú fætt heila gagnasöfn, skjöl eða kóðabrunna inn í Claude fyrir greiningu án þess að þurfa að skipta upp. Leikbreyting fyrir exploratory data analysis og kóða yfirferð.
