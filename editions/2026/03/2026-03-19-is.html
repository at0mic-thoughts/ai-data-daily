📌Fabric Mars 2026 uppfærsla: FabCon útgáfa

Microsoft sleppt sinni stærstu Fabric uppfærslu nokkurn tímann á FabCon Atlanta, spannar stjórnun, data engineering, real-time intelligence, og AI. Helstu hápunktar eru Fabric Runtime 2.0 (Preview), Materialized Lake Views GA með PySpark stuðning, Database Hub fyrir sameinað SQL stjórnun, og branched workspaces með Git integration. Data agents fór líka GA, færir governed AI beint inn í analytics workflow þinn.

🔒Microsoft dýpkar Nvidia samstarf fyrir Enterprise AI
