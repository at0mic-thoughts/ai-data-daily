📌Microsoft Fabric March 2026 Feature Summary and FabCon keynotes

FabCon Atlanta kicked off with a massive update to the Fabric platform. 📌Fabric Data Factory gets Mapping Data Flows and Mirroring expansion

The Data Factory team announced Mapping Data Flows in Fabric (coming June), one of the most requested features for Spark-scale low-code transformations. Additionally, Mirroring for Oracle and SAP Datasphere is now GA, and Mirroring for SharePoint lists is in preview. The Migration Assistant is now in public preview to help teams move ADF and Synapse pipelines to Fabric with minimal friction.

Microsoft Fabric Blog → (Mar 20)

🤖dltHub launches Data Quality framework for Microsoft Fabric

dlthub, the popular open-source Python library for data ingestion, now has official support for Microsoft Fabric. This is critical as Fabric lacks a built-in Data Quality engine. With dlthub, teams can implement Write-Audit-Publish (WAP) patterns, automatic PII detection, and strict schema enforcement directly in Fabric notebooks, significantly reducing operational firefighting.
