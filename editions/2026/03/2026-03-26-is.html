📌Microsoft Fabric stígur inn í „Agentic“ tímabilið með stórum FabCon tilkynningum

Microsoft kynnti nýjar leiðir til að tengja gervigreind við gögn á FabCon 2026. Database Hub (í early access) sameinar stjórnun allra gagnagrunna, Fabric IQ bætir merkingarfræðilegum skilningi við gagnaumhverfið, og Model Context Protocol (MCP) er nú GA, sem gerir AI agentum kleift að nota Fabric tól beint. Þetta er mikilvægt skref frá því að vera bara greiningarvettvangur yfir í sjálfstæðari gervigreindarkerfi sem geta gripið til aðgerða.

📌Fabric Data Factory fær MCP Server og bætta dbt samþættingu

Data Factory teymið kynnti nýjan MCP Server sem leyfir agentum eins og Claude eða GitHub Copilot að stjórna pipelines og gagnaflæði beint í gegnum spjall. Einnig er nú hægt að tengja dbt verkefni beint við GitHub í Fabric, og Mapping Data Flows munu keyra natively í Fabric í júní 2026. Þessar uppfærslur miða að því að gera flutning frá Azure Data Factory óaðfinnanlegan.
