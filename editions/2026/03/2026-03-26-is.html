📌Microsoft Fabric March 2026 uppfærsla og FabCon tilkynningar

FabCon ráðstefnan í Atlanta hófst með gríðarlegri uppfærslu á Fabric platforminum. Helstu fréttir eru: OneLake Catalog Govern fyrir admina (GA), Workspace tags (GA), og Branched workspaces með Git integration (Preview). Einnig nýtt Bulk import/export API fyrir item skilgreiningar sem gerir flutning á milli umhverfa mun auðveldari. Þetta er stærsta uppfærsla ársins hingað til og sýnir mikla áherslu á enterprise stjórnun og DevOps mynstur.

📌Fabric Data Factory fær Mapping Data Flows og betri Mirroring

Data Factory teymið kynnti Mapping Data Flows í Fabric (kemur í júní), sem er ein eftirsóttasta viðbótin fyrir Spark-scale low-code umbreytingar. Einnig er Mirroring fyrir Oracle og SAP Datasphere nú GA, og Mirroring fyrir SharePoint lista er í preview. Migration Assistant er nú í public preview til að hjálpa teymum að flytja ADF og Synapse leiðslur yfir í Fabric með lágmarks fyrirhöfn.

🤖dltHub kynnir gæðastýringu fyrir Microsoft Fabric

dlthub, hið vinsæla opna Python safn fyrir gagnaflutninga, hefur nú opinberan stuðning við Microsoft Fabric. Þetta er mikilvægt þar sem Fabric skortir innbyggt gæðakerfi (Data Quality). Með dlthub geta teymi innleitt Write-Audit-Publish (WAP) mynstur, sjálfvirka uppgötvun á viðkvæmum gögnum (PII) og stranga skema-stýringu beint í Fabric notebook, sem dregur verulega úr rekstrarerfiðleikum.
