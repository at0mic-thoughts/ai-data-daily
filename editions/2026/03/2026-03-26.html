⚡ Must Know

📌Microsoft Fabric enters the "Agentic" era with major FabCon announcements

Microsoft unveiled a new vision for connecting AI to enterprise data at FabCon 2026. Key reveals include the Database Hub (early access) for unified cross-engine management, Fabric IQ for baked-in semantic reasoning, and the General Availability of Model Context Protocol (MCP) support. This marks a strategic shift from Fabric being just an analytics lake to a decision surface where autonomous agents can reason over and act on live data within governed boundaries.

Microsoft Azure Blog → (Mar 26, Microsoft)

📌Fabric Data Factory gets MCP Server and enhanced dbt integration

The Data Factory team introduced an official MCP Server, allowing agents like Claude or GitHub Copilot to manage pipelines and dataflows via chat. New dbt job enhancements now support direct GitHub integration, and Mapping Data Flows are confirmed to run natively in Fabric by June 2026. These updates prioritize "feature parity" to make the migration from Azure Data Factory (ADF) a one-click reality for enterprise teams.

Microsoft Fabric Blog → (Mar 26, Microsoft)

📌Databricks Lakewatch SIEM and SQL Serverless advancements

Databricks has launched Lakewatch, an open, agentic SIEM built on its security lakehouse architecture to combat AI-driven threats. Concurrently, Databricks SQL Serverless is now GA across AWS and Azure, and new advanced governance features for Unity Catalog have been introduced to support massive-scale data estates. These moves signal a direct challenge to traditional security vendors and a focus on cost-efficient serverless compute.

TechEdge AI → (Mar 25, TechEdge)
