📌Microsoft Fabric Keynote: Converged Data Platform & AI Hub

Microsoft announced the unification of transactional, operational, and analytical data under a single architecture. Key highlights include the new Database Hub for managing Azure SQL, Cosmos DB, and PostgreSQL from Fabric, plus the release of Fabric Runtime 2.0 (Spark 4.0/Delta 4.0) and general availability of Fabric Data Agents. This marks a major step in Fabric becoming the central control plane for the entire enterprise data estate.

📌Fabric March 2026 Feature Summary: The 'Agentic' Update

The massive monthly update focuses on AI automation with Model Context Protocol (MCP) servers (Local GA, Remote Preview) and Agent Skills for Fabric. Other major additions: Git integration improvements (selective branching), OneLake Catalog Search API, and GA for OneLake File Explorer for Windows.

Fabric Blog → (Mar 27)

📌Unity Catalog Natively Reads from OneLake

In a significant interoperability win, Databricks announced that Unity Catalog can now natively read data from Microsoft OneLake (Public Preview). This enables data teams to access Fabric-resident data directly from Databricks without movement or complex ETL pipelines.
