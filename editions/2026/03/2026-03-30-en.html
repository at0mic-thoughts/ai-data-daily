📌Fabric March 2026: The Enterprise Data AI Platform Takes Shape

Microsoft's latest Fabric update brings major enterprise capabilities: OneLake Catalog security for third-party engines, Custom Live Pools for dedicated Spark resources, and Resource Profiles that automatically optimize configurations based on workload intent. The Resource Profiles feature is particularly interesting—instead of manually tuning Spark settings, you describe your use case and Fabric recommends optimal compute configuration. This is the kind of abstraction that makes data platforms accessible beyond specialists.

🤖Fabric Remote MCP Server: AI Agents Get Direct Data Platform Access

AI agents can now operate directly within Fabric environments through the new cloud-hosted MCP server. No local installation needed—agents authenticate via Entra ID and work within existing RBAC boundaries. Every operation is audit-logged. This enables agents to create workspaces, manage permissions, and work with item definitions. It's a significant step toward AI-native data platform management.
